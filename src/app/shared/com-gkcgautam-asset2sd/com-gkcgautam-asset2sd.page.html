The main use case of this plugin is to read and write the file from&to the device storage. As per Plugin Documentation This Plugin has no maintainence since last 7 years. We have to try different approach.


Replacement

cordova-plugin-file

This plugin implements a File API allowing read/write access to files residing on the device. The File class implements static convenience functions to access files and directories.
